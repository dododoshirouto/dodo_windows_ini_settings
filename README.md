# dodo_windows_ini_settings

初期化したWindowsの設定を自動でいろいろ変えるやつ

## 前提条件

- Windows 10 / 11
- PowerShell を**管理者権限**で実行すること

## 使い方

1. PowerShellを管理者権限で起動する
2. 実行ポリシーを設定する:
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope Process -Force
   ```
3. スクリプトを実行する:
   ```powershell
   .\windows_ini_settings.ps1
   ```
4. 念の為、再起動しておく

## 注意事項

- スクリプト実行前に**システムの復元ポイントが自動作成**されます
- ログファイルは `%TEMP%\ini_settings_log.txt` に保存されます
- 設定値のカスタマイズはスクリプト冒頭の `$Config` ブロックで行えます

## 変更する内容

### 1. マウス
- ポインターの精度を高める（加速）をオフ
- スクロール速度 1行（`$Config.ScrollLines` で変更可）

### 2. タスクバー
- 検索ボックス非表示
- ウィジェット非表示
- タスクビュー非表示
- チャット非表示
- 左寄せにする

### 3. エクスプローラー
- ファイル名拡張子を表示
- 隠しファイルを表示
- 空のドライブを非表示にするチェックを外す
- タイトルバーに完全なパスを表示

### 4. スクリーンショット
- 保存先をデスクトップに変更

### 5. 個人用設定
- ダークモード設定
- 透明効果をオフ
- 背景を黒の単色に
- アニメーション効果をオフ

### 6. 視覚効果カスタム設定
- 「ドラッグ中にウィンドウの内容を表示」「スクリーンフォントを滑らかに」「アイコンの代わりに縮小版を表示する」以外をオフ

### 7. 電源設定
- 「究極のパフォーマンス」があればそれを、なければ「高パフォーマンス」を有効にする
- 画面タイムアウトを無効（「なし」= 0分）にする

### 8. プライバシー設定
- 不要な追跡設定を無効にする

### 9. クリップボード
- 履歴機能を有効にする

### 10. God Mode
- 全設定にアクセスできるフォルダ「GodMode」をデスクトップに作成
