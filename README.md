# dodo_windows_ini_settings
初期化したWindowsの設定を自動でいろいろ変えるやつ

## 使い方

1. Powershellを管理者権限で起動する
2. `Set-ExecutionPolicy RemoteSigned -Scope Process -Force`
3. windows_ini_settings.ps1を実行する
4. 念の為、再起動しておく

## 変更する内容

### マウス
- マウス感度 12
- ポインターの精度を高める オフ
- スクロール速度 1行

### タスクバー
- 検索ボックス非表示
- ウィジェット非表示
- タスクビュー非表示
- チャット非表示
- 左寄せにする

### エクスプローラー
- ファイル名拡張子を表示
- 隠しファイルを表示
- 空のドライブを非表示にするチェックを外す
- タイトルバーに完全なパスを表示
- スクリーンショット 保存先をデスクトップに変更

### 個人用設定
- ダークモード設定
- 透明効果をオフ
- 背景を黒の単色に
- アニメーション効果をオフ

### デザインとパフォーマンスの調整
- 「ドラッグ中にウィンドウの内容を表示」「スクリーンフォントを滑らかに」「アイコンの代わりに縮小版を表示する」以外をオフ

### 電源設定
- 「究極のパフォーマンス」があればそれを、なければ「高パフォーマンス」を有効にする
- 画面タイムアウトを無効（「なし」= 0分）にする

### その他
- プライバシー 不要な追跡設定を無効にする
- クリップボード 履歴機能を有効にする
- 『God Mode』をデスクトップに召喚
